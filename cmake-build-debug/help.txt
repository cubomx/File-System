[create *name* sizeBlock numBlocks] CREATES A NEW ENVIRONMENT.
[open *name*] LOADS AN EXISTING ENVIRONMENT STORED IN THE COMPUTER.
[save] STORES THE CURRENT ENVIRONMENT.
[load *name* *newName*] LOADS A FILE FROM THE COMPUTER INTO THE FILE SYSTEM.
[download *name* *newName*] STORES A FILE FROM THE FILE SYSTEM INTO THE COMPUTER.
[info] SHOWS INFORMATION ABOUT THE ENVIRONMENT.
[details *fileName*] SHOWS METADATA ABOUT CERTAIN FILE.
[ls] LISTS ALL THE FILES STORED IN THE FILE SYSTEM.
